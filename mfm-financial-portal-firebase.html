Firebase Setup Guide

Connect your portal to Firebase for real-time multi-branch data

🔥 How to Connect Firebase (Step-by-Step)

1

Create a Firebase Project

Go to console.firebase.google.com → Click "Add project" → Name it mfm-akure-portal → Follow the setup wizard → Disable Google Analytics (not needed).

2

Get Your Firebase Config

In your project dashboard → Click the web icon (</>) → Register your app as mfm-portal → Copy the firebaseConfig object shown. It contains your apiKey, authDomain, projectId, etc.

3

Update This File

Open mfm-financial-portal.html in a text editor (Notepad++, VS Code) → Find the section labelled 🔥 FIREBASE CONFIGURATION → Replace all YOUR_API_KEY, YOUR_PROJECT_ID etc. with your actual Firebase values.

4

Enable Firestore Database

In Firebase Console → "Firestore Database" → "Create database" → Choose Production mode → Select a region close to Nigeria (e.g. europe-west1) → Click Enable.

5

Set Firestore Security Rules

In Firestore → Rules tab → Replace the rules with the rules below to allow authenticated users to read/write:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /transactions/{doc} {
      allow read, write: if request.auth != null;
    }
  }
}
6

Enable Firebase Authentication

In Firebase Console → "Authentication" → "Get started" → "Email/Password" → Enable it → Save.

7

Create User Accounts for Each Branch

In Firebase Authentication → "Add user" → Create one account per branch using these emails (set your own passwords):

hq@mfmararomi.org okeijebu@mfmararomi.org ijoka@mfmararomi.org shagari@mfmararomi.org aswani@mfmararomi.org obaile@mfmararomi.org ijapo@mfmararomi.org admin@mfmararomi.org
8

Upload to Your Host & Test

Upload mfm-financial-portal.html to your web host (QServers, Hostinger, WhoGoHost etc.) → Log in from two different devices using two different branch accounts → Enter a record on one device → It should appear on the other device's dashboard within seconds.

9

Customise Branch Emails (Optional)

You can change the branch email addresses in the BRANCHES object at the top of the JavaScript code. Just ensure the emails match exactly what you created in Firebase Authentication.

✅ Firebase Free Tier Limits (More Than Enough)

Firebase's free Spark plan includes: 1GB Firestore storage · 50,000 reads/day · 20,000 writes/day · Unlimited Authentication
For a 7-branch church portal recording hundreds of transactions per month, you will never exceed the free tier.
